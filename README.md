# compact-img-web

Lossy compress images in batch

**Remark**:
This package is renamed from compact-img to compact-img-web to avoid name clash with compact-img-cli.

## Installation

```bash
$ npm i -g compact-img-web
```

## Running the app

```bash
$ compact-img-web
```
