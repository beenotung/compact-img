
# compact-img
Lossy compress images in batch

[![npm Package Version](https://img.shields.io/npm/v/compact-img.svg?maxAge=2592000)](https://www.npmjs.com/package/compact-img)

## Installation

```bash
$ npm i -g compact-img
```

## Running the app

```bash
$ compact-img
```
